# Company Strategy & Org Design

## Mission

Best in the world task manager - Personal OS that transforms incoming chaos of tasks, habits, and routines into an ideal balanced flow (Flow). User is the **Strategy**, system is the **Executive Director (COO)**.

## Organizational Design

Two-lane model with clear hierarchy:

- **Product Lane**: Product Manager → (Support Engineer, UI/UX Designer, Product Critic, Technical Writer, User Research Lead)
- **Engineering Lane**: Founding Engineer → (Backend/Frontend Engineers, DevOps, Security, QA, Technical Critic, Systems Analyst)
- **Meta Roles** reporting directly to CEO:
  - Growth Engineer: Throughput bottlenecks and automation
  - Context Keeper: Organizational memory
  - Risk Manager: Strategic risk assessment
  - Organizational Effectiveness Lead: Performance optimization

## CEO KPI Targets

- **Velocity**: Average Lead Time < 4h for standard tasks
- **Fluidity**: Handoff latency < 30m
- **Quality Rate**: >90% pass Technical Critic review without REVISE
- **Proactivity Index**: >30% self-initiated work

## Environment Gates

Dev → Stage → Prod with clear acceptance criteria at each gate.
